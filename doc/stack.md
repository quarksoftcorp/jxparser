이앱은 한국어,미국어등의 다국어를 지원한다

## 기본

- 패키지명,번들id는 "com.daimontech.jxmparser" 이다
- 앱이름은 "JxmParser"
- json,xml,md 파일을 읽고 쓸수있다.
- 위 3개 포맷을 트리 구조로 트리 구조로 보기 쉽게 뷰어처럼 보여준다.
- 트리 구조중 하나를 선택하면 해당 내용을 편집 할수 있다
- 트리 구조를 편집한 결과를 다시 위 3개 포맷으로 저장 할수 있다.
- json,xml,md 파일을 지원하는 프로그램이 있으면 좋겠다.
- 탭으로 json,xml,md를 멀티로 선택가능
- example을선택시 사용 예제를 정리하여 보여준다.
- 자동으로 포맷이 이쁘게 표현된다.(format)
- 편집시 자동으로 탭 위치가 강제된다. (format) (tab indent)

### 기술스택

- 모바일 환경에서는 pwa로 앱처럼 작동
- 로컬 캐시는 mmkv를 이용해 고속으로 처리
- 웹배포후 웹뷰로 결과물인 url을 로드
- **_ 파이어베이스 mcp를 활용해 설정등 자동구성 _**
- AI가 필요할시 서버는 파이어베이스 펑션v2를 사용하고 랭그래프와 구글 버텍스 ai를 활용한다.
- 기술스택은 firebase앱호스팅을쓰고 next.js를 사용하고 상태관리는 Zustand와 Rxjs를 사용한다. 언어는 타입스크립트, 스타일은 타일윈드css 컴포넌트는 함수형 컴포넌트를 사용한다.
- 스토어정보 자동생성과 스크린샷도 자동으로 생성하여 /metadata폴더에 ios,android,web을 구분하여 저장한다.
- 스크린샷은 실제 앱을 구동시켜 스크린샷을 캡쳐한다.
- 파이어베이스 mcp를 활용해서 직접 파이어베이스 프로젝트를 생성하고 관리한다.
- 파이어베이스 Mcp를 활용해서 앱정보를 등록 한다.
- 파이어베이스 애널리틱스 설정을 적용
